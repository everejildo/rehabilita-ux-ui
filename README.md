# Rehabilita – UX/UI & Prototipo Funcional

Rehabilita es un proyecto de diseño UX/UI y prototipado funcional de una aplicación móvil enfocada en la rehabilitación física y el apoyo al bienestar del usuario.

## Objetivo del proyecto
Diseñar una experiencia digital intuitiva que permita a los usuarios:
- Acceder a rutinas de rehabilitación
- Visualizar ejercicios guiados
- Dar seguimiento a su progreso
- Mejorar la experiencia de recuperación mediante una interfaz clara y accesible

## Herramientas utilizadas
- Figma – diseño de interfaces y experiencia de usuario
- Framer – prototipo interactivo y simulación de funcionalidad

## Funcionalidades del prototipo
- Flujo completo de navegación
- Pantallas interactivas
- Animaciones y transiciones
- Simulación de interacción del usuario
